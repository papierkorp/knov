# User Manual

Welcome to the Knowledge Management System user manual!

## Getting Started

New to the system? Start with our [[../getting-started.md|Getting Started Guide]].

## Basic Operations

### Creating Documents

1. Click the "+" button in the header
2. Enter your document title
3. Start writing using Markdown syntax

### Linking Documents

Use double brackets to link to other documents:

- `[[document-name.md]]` - Simple link
- `[[document-name.md|Custom Text]]` - Link with custom text
- `{{document-name.md}}` - Alternative link syntax

### Organizing Content

- Use folders to group related documents
- Add tags for better categorization
- Set project associations for tracking

## Advanced Features

### Search and Filtering

- Use the search bar for quick text search
- Apply metadata filters for precise results
- Combine multiple criteria with AND/OR logic

### Collaboration

- Track document history with Git integration
- See recent changes in the activity feed
- Comment and discuss using linked notes

## Tips and Tricks

1. Use consistent naming conventions
2. Link related documents liberally
3. Keep metadata updated for better organization
